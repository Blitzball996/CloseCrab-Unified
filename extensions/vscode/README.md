# CloseCrab AI - VS Code Extension

AI coding assistant powered by CloseCrab-Unified.

## Features
- Chat sidebar with AI
- Explain/Fix/Refactor selected code
- Generate tests
- Code review

## Requirements
- CloseCrab-Unified running (`closecrab-unified.exe`)
- Backend accessible at http://localhost:9001

## Usage
1. Start CloseCrab-Unified
2. Open VS Code
3. Click the CloseCrab icon in the activity bar
4. Start chatting!

## Commands
- `CloseCrab: Open Chat` - Open chat panel
- `CloseCrab: Explain Selection` - Explain selected code
- `CloseCrab: Fix Selection` - Fix bugs in selection
- `CloseCrab: Refactor Selection` - Improve code quality
- `CloseCrab: Generate Tests` - Create unit tests
- `CloseCrab: Review File` - Full file review
