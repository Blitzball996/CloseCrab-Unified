# CloseCrab AI - Sublime Text Plugin

## Install
Copy this folder to your Sublime Text Packages directory:
- Windows: `%APPDATA%\Sublime Text\Packages\CloseCrab\`
- macOS: `~/Library/Application Support/Sublime Text/Packages/CloseCrab/`
- Linux: `~/.config/sublime-text/Packages/CloseCrab/`

## Usage
- `Ctrl+Shift+C` - Open chat
- `Ctrl+Shift+E` - Explain selection
- `Ctrl+Shift+F` - Fix selection
- Command Palette: "CloseCrab: ..."

## Requirements
- CloseCrab-Unified running on localhost:9001
