# 民事起诉状/反诉状撰写技能

基于实战验证的起诉状写作方法。核心：诉请明确、事实清晰、证据充分、法律适用准确。

## 格式规范

```
民事起诉状（或：反诉状）

原告（反诉人）：[姓名/公司名]，[基本信息]
被告（被反诉人）：[姓名/公司名]，[基本信息]

诉讼请求：
一、请求判令被告支付[具体款项] [金额]元；
二、请求判令被告[具体行为]；
三、请求判令被告承担本案诉讼费、保全费等。

事实与理由：
[按时间线详细陈述]

证据：
[证据清单简要列举]

此致
[法院名称]
原告（反诉人）：
[日期]
```

## 诉讼请求写作规则

### 必须具备的特征
- **明确性**：具体到金额数字、行为内容
- **可执行性**：法院判决后能直接执行
- **合法性**：有法律依据支撑

### 常见请求类型
1. 金钱给付：支付工资/报酬/赔偿金 + 具体金额
2. 行为请求：停止侵害/赔礼道歉/恢复原状
3. 确认请求：确认合同无效/确认劳动关系
4. 附带请求：诉讼费由被告承担

### 反诉特有请求
- 追究对方指使证人作伪证的法律责任
- 要求支付劳动报酬（同工同酬原则）
- 精神损害赔偿
- 要求法院将涉嫌犯罪的移交有关部门

## 事实与理由写作

### 时间线叙事法
按时间顺序还原事实，每个关键节点标注日期：
"2022年1月5日，原告...；1月6日第一天，...；1月7日早上，..."

### 证据穿插引用
在事实陈述中直接引用证据：
"根据证人电话录音16：52可知，不让车手开的原因为..."
"从《车辆使用合同》3的第一条可知..."

### 因果关系论证
事实 → 对方过错 → 损害结果 → 因果关系 → 法律责任

### 对方恶意行为揭露
- 哄骗/欺诈行为
- 隐瞒关键信息
- 恶意串通第三方
- 伪造/篡改证据

## 法律依据引用

### 格式
"根据《中华人民共和国民法典》第[条]条规定，[条文内容概述]。"

### 常用法条
- 民法典第1192条（个人劳务/追偿权）
- 民法典第1198条（安全保障义务）
- 民法典第509条（合同履行）
- 劳动合同法第3条（诚实信用原则）
- 刑法第307条（妨害作证罪）
- 刑法第307条之一（虚假诉讼罪）

## 反诉的特殊技巧

### 何时提起反诉
- 对方存在违约在先
- 对方行为涉嫌犯罪
- 己方有独立的请求权

### 反诉与答辩的区别
- 答辩：否定对方请求（防守）
- 反诉：提出己方独立请求（进攻）
- 可以同时提交

## 语言风格
- 事实部分：客观、具体、有时间线
- 理由部分：逻辑严密、法条引用准确
- 结论部分：明确、有力
- 整体：严谨但有说服力，让法官看到事实全貌

## 避免的错误
- 诉讼请求模糊（如"赔偿损失"不写金额）
- 事实陈述带过多情绪
- 法条引用错误或过时
- 遗漏关键时间节点
- 证据与事实脱节
