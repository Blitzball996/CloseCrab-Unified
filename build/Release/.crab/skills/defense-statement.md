# 民事答辩状撰写技能

基于实战验证的答辩状写作方法论。核心原则：逻辑严密、证据充分、攻防兼备。

## 格式规范

```
答辩状

答辩人：[姓名]，[性别]，[出生日期]，[民族]，公民身份号码：[号码]，现住[地址]，联系电话：[电话]。
被答辩人：[公司名/姓名]，住所地[地址]，统一社会信用代码：[代码]。
    法定代表人：[姓名]，职务：[职务]。

[答辩人]就[被答辩人]诉答辩人[案由]一案的诉讼请求及事实与理由做如下答辩：

一、[第一个答辩要点标题]
[论述内容]

二、[第二个答辩要点标题]
[论述内容]

...

综上，[被答辩人]的诉求缺乏相应的事实和法律依据，恳请人民法院查明案件事实，依法予以驳回。

此致
[法院名称]
答辩人：
[日期]
```

## 写作策略

### 1. 开篇定调
- 首先质疑原告的请求权基础："原告在诉状中并没有明确表述其与被告之间是何种法律关系，依据何种法律关系向被告主张权利"
- 要求原告明确法律关系，掌握主动权

### 2. 逐条反驳结构
每个答辩要点应包含：
- **标题**：用一句话概括核心观点（如"答辩人不存在重大过失和故意，不应承担赔偿责任"）
- **事实陈述**：按时间线还原真实情况，引用具体证据
- **证据引用**：精确到录音时间戳（如"证人电话录音21分13秒"）、文件名称、条款编号
- **法律依据**：引用具体法条（如"根据民法典第一千一百九十二条规定"）
- **逻辑推导**：从事实+法律得出结论

### 3. 证据运用技巧
- 录音证据：精确引用时间戳和原话，用引号标注
- 书面证据：指出矛盾之处（如"付款凭证备注是项目款而非车损赔偿款"）
- 对方证据反用：用对方提交的证据证明己方观点
- 证人证言：引用证人确认的关键事实

### 4. 攻防兼备
- 防守：逐条否定原告的事实主张和法律依据
- 进攻：指出原告的违法行为（伪造证据、虚假诉讼、恶意串通）
- 反诉预告：保留向原告索赔的权利

### 5. 金额反驳
- 质疑金额的计算依据和认定机构
- 区分直接损失和间接损失
- 指出已由保险/第三方赔付的部分
- 质疑付款凭证的备注与诉请不符

### 6. 程序性抗辩（可选）
- 管辖权异议
- 适用程序异议（简易→普通）
- 诉讼时效抗辩
- 请求权竞合

## 语言风格
- 用词严谨有力："捏造事实"、"恶意串通"、"虚构债务"、"子虚乌有"
- 逻辑连接词：首先、其次、再者、综上
- 反问增强说服力："原告否定在拍摄，那为何要看画面？自相矛盾。"
- 避免情绪化，保持法律专业性

## 常用法条
- 民法典第1192条（雇主责任/追偿权）
- 民法典第509条（合同履行原则）
- 民法典第1032-1033条（隐私权）
- 民事诉讼法第111条（伪造证据）
- 劳动合同法第3条、第11条
- 道路交通安全法第22条

## 避免的错误
- 不要笼统概括，每个论点必须有具体证据支撑
- 不要遗漏对方证据中的矛盾点
- 不要只防守不进攻
- 不要使用模糊的时间表述（用具体日期）
- 不要忽略程序性问题
