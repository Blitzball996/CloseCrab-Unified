# 劳动仲裁申请书撰写技能

基于实战验证的仲裁申请书写作方法。核心：请求明确、金额精准、依据充分。

## 格式规范

```
劳动仲裁申请书

申请人：[姓名]，[性别]，[出生日期]，[民族]，公民身份号码：[号码]，现住[地址]，联系电话：[电话]
被申请人：[公司全称]。住所地：[注册地址]。实际经营地址：[实际地址]。统一社会信用代码：[代码]。法定代表人：[姓名]。职务：[职务]。联系电话：[电话]

仲裁请求：
一、请求裁决被申请人支付申请人[具体款项名称]人民币 [金额] 元。
二、请求裁决被申请人支付[具体款项名称] [金额]元。
以上各项合计金额：人民币[总额]元。

事实和理由：
[详细陈述]

综上所述，被申请人的行为已经严重侵害申请人的合法权益，为了维护申请人的合法权益，申请人根据《劳动法》、《劳动合同法》等相关规定，特向贵委申请劳动仲裁，请求贵委依法裁决！

此致
[区]劳动人事争议仲裁委员会
申请人：
[日期]
```

## 仲裁请求类型

### 常见请求项
1. 解除劳动合同经济补偿金（N或N+1）
2. 未签劳动合同双倍工资差额
3. 拖欠工资/工资差额
4. 加班费
5. 未休年假工资
6. 违法解除赔偿金（2N）
7. 社保/公积金补缴

### 金额计算
- 经济补偿金 = 月工资 × 工作年限（每满一年1个月，6个月以上不满一年按1年，不满6个月按0.5个月）
- 双倍工资 = 月工资 × 未签合同月数（最多11个月）
- 违法解除赔偿金 = 经济补偿金 × 2

### 精确到分
金额必须精确计算，如"536.02元"而非"约500元"。列出计算过程。

## 事实与理由写作

### 结构
1. 入职时间、岗位、合同签订情况
2. 争议事实（按时间线）
3. 对方违法/违约行为
4. 己方权益受损情况
5. 法律依据

### 关键要素
- 入职日期精确到日
- 离职日期和方式（主动/被迫/违法解除）
- 工资标准（月薪/日薪）
- 书面证据引用（解除通知书、工资条、合同条款）

## 注意事项
- 被申请人信息要完整（注册地+实际经营地都写）
- 仲裁请求要具体可执行，不能模糊
- 每项请求单独列出，最后写合计
- 事实陈述客观简洁，不带情绪
- 法律依据写在最后一段
- 管辖：一般由用人单位所在地或劳动合同履行地仲裁委管辖

## 常用法条
- 劳动合同法第38条（被迫解除）
- 劳动合同法第46条（经济补偿情形）
- 劳动合同法第47条（经济补偿计算）
- 劳动合同法第82条（双倍工资）
- 劳动合同法第87条（违法解除赔偿）
- 劳动法第50条（工资支付）
