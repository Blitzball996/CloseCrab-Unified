# 证据目录撰写技能

基于实战验证的证据目录写作方法。核心：编号清晰、证明目的精准、逻辑链完整。

## 格式规范

```
证据材料清单

案号：[案号]
当事人：[原告/被告姓名]

| 序号 | 证据名称 | 证据类型 | 证明目的 | 页码 | 备注 |
|------|---------|---------|---------|------|------|
| 1 | ... | 书证 | ... | 1-3 | 原件/复印件 |
| 2 | ... | 电子证据 | ... | 4-8 | 公证件 |
| ... | ... | ... | ... | ... | ... |

以上证据共[N]份，[M]页。

提交人：
日期：
```

## 证据类型分类

| 类型 | 示例 |
|------|------|
| 书证 | 合同、协议、通知书、收据 |
| 物证 | 实物、照片 |
| 视听资料 | 录音、录像、监控 |
| 电子证据 | 微信聊天记录、邮件、转账记录 |
| 证人证言 | 证人书面陈述 |
| 鉴定意见 | 司法鉴定报告 |
| 勘验笔录 | 现场勘验记录 |

## 证明目的写作规则

### 好的证明目的（精准、具体）
- "证明原告明确约定被告工作范围仅为送车，而非参与拍摄"
- "证明原告指使证人作伪证，录音00:54处证人明确表示'他给我写了一份让我照着抄'"
- "证明车辆损失已由保险公司全额赔付，原告无权再向被告追偿"
- "证明双方于2026年3月3日签订解除劳动合同通知书，确认补偿金22500元"

### 差的证明目的（避免）
- "证明相关事实"（太笼统）
- "证明被告有过错"（没有具体指向）
- "证明原告的主张"（循环论证）
- "供法庭参考"（无意义）

## 证据编排策略

### 按逻辑链排列
1. 基础关系证据（合同、身份证明）
2. 核心事实证据（关键行为的直接证据）
3. 对方过错证据（违约/违法行为证据）
4. 损害结果证据（损失金额、伤害程度）
5. 因果关系证据（连接过错与损害）
6. 辅助证据（补强前述证据）

### 录音证据的特殊处理
- 提供录音文字转写稿（标注时间戳）
- 证明目的引用具体时间点和原话
- 说明录音的合法性（非窃听，当事人之一在场）
- 如有必要，申请声纹鉴定

### 微信/电子证据
- 截图需完整（显示对方头像、昵称、时间）
- 最好做公证保全
- 证明目的要指出具体哪条消息证明什么

## 反面教材（避免）

以下是实际案例中被法官批评的证据目录写法：
- 证据名称写"相关材料"——不知道是什么
- 证明目的写"证明被告存在过错"——没说什么过错
- 多份证据合并一个编号——法官找不到
- 页码标注错误——影响庭审效率
- 证据类型标注错误（把电子证据标为书证）

## 注意事项
- 每份证据独立编号，不要合并
- 页码连续标注，方便法官翻阅
- 原件/复印件必须标注
- 录音证据附文字版转写
- 外文证据附翻译件
- 证据份数：法院1份 + 对方当事人各1份 + 自留1份
